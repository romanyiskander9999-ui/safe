# CHANGELOG

## [0.1.0] - 2026-06-12

### مضاف
- ✨ إطار عمل المشروع الأساسي مع Next.js 14
- 🎨 نظام التصميم الشامل مع Tailwind CSS
- 🔐 نظام المصادقة مع دعم ثلاثة أدوار (ولي أمر، طفل، مدرسة)
- 📊 لوحة تحكم الوالد مع إحصائيات شاملة
- 👧 لوحة تحكم الطفل مع واجهة ودية وممتعة
- 🏫 لوحة تحكم المدرسة لتتبع الطلاب
- ⏱️ نظام إدارة وقت الشاشة المتقدم
- 🔒 مرشح المحتوى مع تصنيفات الأمان
- 📈 نظام التقارير المفصل مع رسوم بيانية
- 📚 دروس التوعية الرقمية
- 🔔 نظام التنبيهات والإشعارات
- 👨‍👩‍👧‍👦 إدارة الأطفال المتعددين
- 💬 صفحة الدعم والمساعدة
- 🌍 دعم كامل للعربية وRTL
- 🎯 نظام إدارة الحالة مع Zustand
- 📱 واجهة مستجيبة وودية

### المكونات المضافة
- LoginPage: صفحة تسجيل الدخول
- Navbar: شريط التنقل
- StatCard: بطاقة الإحصائيات
- AlertBadge: شارة التنبيهات
- RiskBadge: شارة مستوى الخطر
- ScreenTimeCard: بطاقة وقت الشاشة
- AlertsCard: بطاقة التنبيهات
- AppUsageCard: بطاقة استخدام التطبيقات
- QuickActionsCard: بطاقة الإجراءات السريعة

### الصفحات المضافة
- `/` - الصفحة الرئيسية (توجيه ديناميكي حسب الدور)
- `/settings/screen-time` - إعدادات وقت الشاشة
- `/content-filter` - مرشح المحتوى
- `/reports` - التقارير
- `/awareness` - دروس التوعية
- `/children` - إدارة الأطفال
- `/support` - الدعم والمساعدة

### متاجر Zustand
- `useAuthStore` - إدارة المصادقة وبيانات المستخدم
- `useDataStore` - إدارة بيانات التطبيق
