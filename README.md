## 📋 نظرة عامة

SafePlay منصة شاملة لحماية الأطفال في البيئة الرقمية. توفر أدوات متقدمة للآباء والمدارس لمراقبة ومراقبة استخدام الأجهزة والتطبيقات، مع التركيز على التوعية الرقمية والسلامة الإلكترونية.

### المميزات الرئيسية ✨

- **إدارة وقت الشاشة** ⏱️
  - تحديد حدود يومية وأسبوعية
  - حجب تلقائي عند الوصول للحد
  - تنبيهات قبل الوصول للحد
  - حجب وقت النوم

- **مرشح المحتوى** 🔒
  - تصنيف التطبيقات حسب مستوى الأمان (آمن، يحتاج إشراف، محظور)
  - إدارة التطبيقات المسموحة والمحظورة
  - حماية من المحتوى الضار

- **لوحات التحكم المخصصة** 📊
  - لوحة الوالد: إدارة شاملة لأطفال متعددين
  - لوحة الطفل: واجهة صديقة وممتعة مع نظام نقاط
  - لوحة المدرسة: تتبع أداء الطلاب

- **التقارير المفصلة** 📈
  - تقارير أسبوعية وشهرية
  - تحليل استخدام التطبيقات
  - رسوم بيانية توضيحية
  - تحميل التقارير بصيغة PDF

- **دروس التوعية** 📚
  - محتوى تعليمي حول الأمان الرقمي
  - دروس مخصصة حسب العمر
  - اختبارات ونقاط مكتسبة
  - موضوعات متنوعة (التنمر، الخصوصية، المحتوى المزيف، إلخ)

- **نظام التنبيهات** 🔔
  - تنبيهات فورية لمحاولات الوصول للتطبيقات المحظورة
  - تنبيهات للتجاوزات الزمنية
  - إشعارات النجاح والإنجازات

- **دعم RTL الكامل** 🌍
  - واجهة مصممة للعربية
  - دعم كامل للاتجاه من اليمين لليسار
  - نصوص عربية احترافية

## 🚀 البدء السريع

### المتطلبات الأساسية
```bash
- Node.js 18+ 
- npm أو yarn أو pnpm
```

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/romanyiskander9999-ui/safe.git
cd safe

# تثبيت الحزم
npm install
# أو
yarn install
```

### التطوير

```bash
# تشغيل خادم التطوير
npm run dev
# أو
yarn dev

# الوصول إلى التطبيق
# http://localhost:3000
```

### الإنتاج

```bash
# بناء المشروع
npm run build

# تشغيل الخادم
npm run start
```

## 📖 أدلة الاستخدام

### لولي الأمر 👨‍👩‍👧‍👦

#### الوصول إلى اللوحة الرئيسية
1. قم بتسجيل الدخول باستخدام بيانات اعتماد ولي الأمر
2. شاهد ملخص استخدام شاشة أطفالك
3. اعرض التنبيهات والتحذيرات

#### إدارة وقت الشاشة
1. انتقل إلى "الإعدادات" → "وقت الشاشة"
2. عيّن الحدود اليومية والأسبوعية
3. فعّل حجب وقت النوم
4. اضبط الحدود حسب الفئات
5. احفظ الإعدادات

#### مراقبة التطبيقات
1. انتقل إلى "مرشح المحتوى"
2. اعرض جميع التطبيقات وتصنيفات الأمان
3. احظر أو السماح بالتطبيقات
4. اعرض سبب الحظر

#### عرض التقارير
1. انتقل إلى "التقارير"
2. شاهد الإحصائيات الأسبوعية
3. اعرض أعلى التطبيقات المستخدمة
4. حمّل التقرير بصيغة PDF

### للطفل 👧👦

#### معرفة الوقت المتبقي
1. انظر إلى الدائرة الكبيرة في الصفحة الرئيسية
2. ستعرف الدقائق المتبقية
3. ستحصل على تنبيهات قبل النهاية

#### كسب النقاط والشارات
1. أكمل المهام اليومية
2. شارك في دروس التوعية
3. اكسب نقاطاً وشارات
4. شاهد إنجازاتك

#### تعلم الأمان الرقمي
1. انتقل إلى "الدروس"
2. اختر درساً يناسب عمرك
3. اقرأ المحتوى التعليمي
4. أجب على الأسئلة
5. احصل على شهادة الإكمال

### للمدرسة 🏫

#### مراقبة الصف
1. تسجيل الدخول كمدرسة
2. اعرض قائمة الطلاب
3. شاهد إحصائيات الاستخدام لكل طالب
4. تتبع إكمال الدروس

#### التواصل مع الأهالي
1. أرسل تقارير شاملة
2. شارك الملاحظات والتنبيهات
3. تعاون على حماية الأطفال

## 📝 الملفات والمجلدات المهمة

### `store/authStore.ts`
متجر المصادقة الذي يدير:
- بيانات المستخدم الحالي
- تسجيل الدخول والخروج
- بيانات اعتماد التطوير

### `store/dataStore.ts`
متجر البيانات الذي يحتوي على:
- قائمة الأطفال
- إعدادات وقت الشاشة
- التطبيقات والتقييمات
- التنبيهات والأنشطة

### `types/index.ts`
تعريفات TypeScript الشاملة لجميع أنواع البيانات.
